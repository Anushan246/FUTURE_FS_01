My Projects

Project 1: Student Enrollment System

A mini DBMS project that manages student enrollment data. This system allows for efficient storage, retrieval, and management of student information, course registrations, and academic records. Implemented using SQL for database operations and a user-friendly interface for easy data manipulation.

Project 2: Color Audio Guide for Visually Challenged

Developed a Python-based application that assists visually impaired individuals in identifying colors. The system uses computer vision to detect colors in the environment and provides audio feedback. It utilizes CSV files for mapping color codes to descriptive names, enhancing the user's understanding of their surroundings.

Project 3: Face Awareness System for Blind

Created an innovative Python application to help blind individuals recognize faces. This system employs facial recognition technology and XML files for storing facial features. It captures images through a camera, processes them to identify faces, and provides audio output to describe the detected individuals, improving social interaction for the visually impaired.
